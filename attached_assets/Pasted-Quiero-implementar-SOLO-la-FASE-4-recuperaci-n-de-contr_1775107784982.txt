==================================================
PARTE 1 — RECUPERAR CONTRASEÑA
==================================================

FLUJO:

1. En login agregar botón:
   "¿Olvidaste tu contraseña?"

2. Al hacer clic:
   pantalla donde usuario ingresa su email

3. Backend:
   - validar si el email existe
   - SI existe → generar token seguro (random + expiración 15-30 min)
   - guardar token en DB (o tabla password_reset_tokens)

4. Enviar email con link:
   ejemplo:
   https://tuapp.com/reset-password?token=XYZ

5. Pantalla reset:
   - validar token
   - pedir:
     nueva contraseña
     confirmar contraseña

6. Backend:
   - validar token válido y no expirado
   - actualizar contraseña
   - invalidar token
   - permitir login normal

REGLAS:
- No revelar si el email existe o no (seguridad)
- Mensaje genérico tipo:
  "Si el correo existe, te enviamos instrucciones"
- Token de un solo uso
- Token expira automáticamente
- Contraseñas deben validarse (mínimo 6-8 caracteres)
