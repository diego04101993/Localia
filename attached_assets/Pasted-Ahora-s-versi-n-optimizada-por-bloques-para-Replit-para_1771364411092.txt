PROMPT Replit — FASE 1 (Base sólida)
Crea un proyecto fullstack con:
•	Next.js (App Router) + TypeScript + Tailwind + shadcn/ui
•	PostgreSQL + Prisma
•	Auth email/password
•	Roles: SUPER_ADMIN, BRANCH_ADMIN, CUSTOMER
Requisitos:
1.	Rutas:
•	/superadmin (solo SUPER_ADMIN)
•	/dashboard (solo BRANCH_ADMIN)
•	/app/[slug] (vista pública PWA de sucursal)
2.	DB models mínimos Prisma:
•	User(id, email, passwordHash, role, branchId nullable)
•	Branch(id, name, slug unique, status active|suspended|blacklisted, createdAt)
3.	Superadmin UI:
•	lista de sucursales
•	crear sucursal (name, slug)
•	cambiar status (active/suspended/blacklisted)
4.	Middleware de seguridad:
•	si branch status != active, bloquear acceso a dashboard y app y mostrar pantalla “Servicio no activo”
5.	Seed:
•	1 superadmin
•	1 branch demo “Box Central”
•	1 branch admin
Entrega:
•	README con pasos
•	.env.example
•	prisma migrate + prisma seed funcionando

PROMPT Replit — FASE 3 (Paquetes + Reservas + Descuento automático)
Agregar:
1.	Modelos Prisma:
•	Package(id, branchId, name, type classes|unlimited, totalClasses optional)
•	CustomerPackage(id, customerId, packageId, remainingClasses, status, startDate, expiryDate optional)
•	Booking(id, branchId, customerId, startsAt, endsAt, type class|appointment, status booked|cancelled|attended|no_show, cancelTime optional, checkinTime optional, deductedAt optional)
•	BranchSettings(branchId, graceMinutes default 1, lateCancelDeducts default true)
2.	App cliente:
•	pantalla Agenda (calendario + elegir horario)
•	crear booking si tiene clases disponibles (si package type=unlimited, permitir)
•	cancelar booking antes de inicio sin descuento
3.	Dashboard sucursal:
•	lista de reservas
•	botón check-in
4.	Descuento automático:
•	función idempotente processPendingBookings(branchId):
o	si booking ya pasó + graceMinutes y status booked:
	si cancelTime < startsAt => no descuenta (status cancelled)
	si no canceló => descuenta 1 clase (si type=class y package classes) y marca deductedAt
•	Ejecutar processPendingBookings al entrar al dashboard y con endpoint /api/jobs/process (protegido)
